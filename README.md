# macbook-provisioning

## 実行
```
$ git clone git@github.com:chocopie116/macbook-provisioning.git
$ export PATH=$PATH:/opt/homebrew/bin
$ make setup
$ make package/install
```

## 手動インストールが必要なアプリ
- [InYourFace](https://inyourface.app/) - カレンダー通知アプリ
